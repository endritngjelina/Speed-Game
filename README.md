# Speed‑Game

Speed‑Game is a simple, fast‑paced browser game designed to test and improve your reaction time. The project includes multiple versions of the game (`speed game 0.html` through `speed game 4.html`), each representing a step in the development process — from a basic clickable box to more advanced interactions and visual feedback.

---

## 🚀 Features

- Multiple versions showing incremental improvements  
- Pure HTML, CSS, and JavaScript — no external libraries  
- Reaction‑time measurement  
- Visual cues and interactive elements  
- Beginner‑friendly code structure

## 🎮 How to Play

1. Open any of the `speed game X.html` files in your browser.  
2. Wait for the game element to appear or change color.  
3. Click as fast as you can.  
4. Your reaction time will be displayed on screen.
